## How to install and run
____
### Clone the repository
 
```sh
$ cmd
$ git clone https://github.com/NicolasIDias/NotePad
$ cd NotePad
```
 
### Install the necessary packages (Install dependencies)
```sh
$   pip install -r requirements.txt

```

## Usage
To launch the GUI:
```
python3 -m main
```
